# project-galaxy


## :rocket: Development

## To start the webpack:

`cd assets`
`npm run dev`

## To stop the webpack

`Ctr + C`

## Create a branch

#### If you are in assets directory, come out using 
`cd ..`

#### Create a branch
`git checkout -b feature/header`

## To Pull changes
`git pull orgin branch-name`

## Push the code
`git add .`
`git commit -m "Your Message"`
`git push origin feature/branch-name`
